Version 1.0.0.98 | 2023-01-27 |

Published to Early Release Users ONLY If you notice any issues with the Extension please report it ASAP so that we can fix it! Thanks! Please Report issues here: https://support.ezbookprep.com

  • New Features:
    • - None (maintanance release)

  • Improved features:
    • Order Processing has been improved to handle certain issues tht caused it to hang.
    • The "Can I List it" and "Inventory Boxes" Position "memory" has been improved.
    • Improved interaction with other extensions, should limit the occasional weird page.
    • Added more process info to the console

  • Bug Fixes:
    • The "Reset All Defaults" Button in settings Page works on first click.
    • Floating Charting window now resizes correctly.
    • Other internal bugs squashed
    • Version # now appears on the Relase Notes Page

  • Known Issues: ( Which should be fixed soon! )
    • Turning off the Charts, currently blocks all other Offer Side Bar Modifications, that hide certain sections.
    • (Personal Accounts Only) An intermittent error on Amazon's "Your Orders" Page, causes our "Order Data" button to not appear for all orders. Can be fixed by clicking 'Buy Again" then the 'Orders' tabs or sometimes by reloading the page.
    • Pages with URLs that contain "AOD=1" behave inconsistently. Looking like it is being caused by another extension, this is still being determined. You can try turning off some of your charting Extensions or disabling our floating charts.

  • NOTE(S):
    • Inventory Box has been disabled (Feature being rebuilt)
    • If you think that our EasyBookPrep extension is causing any issues, go to our Extension settings and turn that section off, AND RELOAD ALL AMAZON Buying Pages, if your still having issues it is most likely not our extension. You can also disable our extension to test, if needed.
      • If you still think it is an issue with our extension please open a ticket at our support Portal: https://support.ezbookprep.com
    • We can set you up to manually submit Order Data via an FTP, email support for access.

    Coming Soon::
    + Self Configuration of Extension
    + Reminders/Scheduled Order Processing

Version 1.0.0.93 | 2023-01-20 |

  • New Features:
    • The "Can I List it" and "Inventory Boxes" can now be placed differently on Product Info & another when the Offers are Showing.
    • The width of the Offer Sidebar can now be adjusted in Settings
      ( Under Charting Section, Only available when the charts are On. )

  • Improved features:
    • Turning Off Charts, Stops our extension from modifying the Amazon Product and Offers Side Bar, preventing any Conflicts with other extensions or any Amazon changes that may occur. Use as needed.
      • Note: "Can I List It" and "Inventory" Floating Boxes still work but can be turned off individually in our settings, if needed.
    • Expanded our Order Updates to 12 weeks of historical Order Data. So,If your behind processing your orders, Just process your orders and we get/Update all orders 12 weeks back.
    • Setting Page has been updated to clarify what setting are available when things are turned off.

  • Bug Fixes:
    • Floating window(s) should appear and disappear more consistently
    • Other smaller Technical bugs where squashed & cleaned up
    • Updated the "Help and Training Login:" Link in the Extension popup

  • Known Issues: ( Which should be fixed soon! )
    • Turning off the Charts, currently blocks all other Offer Side Bar Modifications, that hide certain sections.
    • The "Reset All Defaults" Button needs to be pressed Twice for the defaults to be applied.
    • Inventory Box will not show Data (Feature being rebuilt), to be more stable and update order data faster for more up-to-date information.
    • (Personal Accounts Only) An intermittent error on Amazon's "Your Orders" Page, causes our "Order Data" button to not appear for all orders. Can be fixed by clicking 'Buy Again" then the 'Orders' tabs or sometimes by reloading the page.
    • Pages with URLs that contain "AOD=1" behave inconsistently. Looking like it is being caused by another extension, this is still being determined. You can try turning off some of your charting Extensions.
    • There is a bug in the charting window that prevents the window from going larger than a certain size.
